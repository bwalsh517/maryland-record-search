Maryland Birth & Death Record Search

A genealogy research tool for locating historical Maryland birth and death certificate scans.

This page is a reference implementation showing how to call the MDRecordSearch library from your own site's search UI. See app.js for the function calls that do all the work: MDRecordSearch.lookup(...), MDRecordSearch.lookupSeries(...), and MDRecordSearch.lookupCertificate(...).
